Описание и возможности

Filters — это онлайн редактор изображений, позволяющий быстро и просто провести ретушь изображения с помощью базовых функций.
С помощью фоторедактора Filters можно изменять яркость, контрастность и насыщенность изображения.
Функции «шум» и «сепия» позволят отредактировать ваши фотографии в винтажном стиле.

Наведите на изображение, чтобы увидеть фильтры в действии.

Процесс создания

Первым делом нужно было установить операционную систему Ubuntu с помощью виртуальной машины VirtualBox, чтобы в дальнейшем упростить работу с Docker и Flask приложением. В качестве программы для написания кода использовалась Visual Studio Code.
Затем установили и настроили Docker и Flask приложение, используемый образ для развертывания Flask приложения в Docker — tiangolo/uwsgi-nginx-flask.
Flask приложение содержит шаблон, отображающий статическое и динамическое содержимое для пользователей. Выполнили дизайн и вёрстку одностраничного сайта с использованием HTML-кода, CSS-стилей и JavaScript-кода.

Более подробную инофрмацию процесса создания можно найти в соответствующих блоках.

VirtualBox

Запуск операционной системы Ubuntu 22.10 с помощью виртуальной машины VirtualBox.

Docker

Установка Docker.

Настройка Flask приложения:
__init__.py — создание экземпляра Flask и импортирование логики из views.py.
views.py — открытие веб-страницы.
uwsgi.ini — содержит конфигурацию uWSGI для Flask приложения.
requirements.txt — указание зависимостей, которые pip менеджер пакетов устанавливает при развертывании Docker.
main.py — точка входа в приложение.
Dockerfile — команды, которые определяют, как будет создаваться образ и какие дополнительные требования будут включены.

Настройка Docker:
start.sh — сценарий оболочки, который создает образ и контейнер Docker из Dockerfile.
Используемый образ — tiangolo/uwsgi-nginx-flask.

CSS/HTML
